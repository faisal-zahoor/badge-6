for example: 

            h1 {
                font-family: "xyx";

Select By Class

we should give the name of class with preceding dot(.) and apply this
for example: 

        .green {
            background-color: green;
            font-size: 30px;
            padding: 10px 20px;
        }

we should give the id of element with preceding dot(#) and apply this
for example: 

            #black-btn {
                background-color: black;
                color: #fff;
            }
